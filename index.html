CSS Color Selection

CSS Color Declarations

Declaration Name Description
color: Foreground color Primary color of the selected component. Colors the foreground. Examples include text and full item block.
background-color: Background color Color appearing behind the focal point of the object. Examples include coloring behind text and images.

CSS Color Value Options

Value Example Name Description
blue Color Name Keywords Named colors that are pre-approved to provide an exact color. In the example, the value will provide a typical blue color.
#ff0000 Hexadecimal Colors Hexadecimal notation representing a color. Each group of 2 hex values represents a color in the red/green/blue color set. In the example, the text would be all red.
rgb(12, 201, 56) rgb() Colors Declares a color with a value for red, green, and blue between 0-255. Similar to hex, but more common. The example is this color.
hsl(230, 70%, 50%) hsl() Colors Declares a color using hue (ranging from 0 to 360 color wheel), saturation % (how grey it is), and lightness % (how bright it is). The example is this color.
rgba(240, 30, 100, 0.5) Alpha Color Values Adds an 'Alpha Value' to rgb or hsl color declarations. HSL format is hsla(#,%,%,a). The alpha value controls the opacity of the color and is between 0 and 1. The example is this color.
